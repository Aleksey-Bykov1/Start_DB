
    В рамках курсового проекта по релязионным базам данных создаю базу данных учебного заведения "College".
Основные цели базы данных:
1. Создание списка студентов проходящих обучение с внесением личных данных и идентификаторов.
2. Создание списка преподавателей и предметов которые они будут вести.
3. Так же списка факультетов на которые будут распределены студенты.
4. Учет оценок полученных студентами за экзамены.
5. Создание перечня библиотечного фонда учебного заведения.
6. Учет полученных в личное пользование книг студентами.

Описывать структуру каждой таблици не стал так как при создании таблиц на каждое поле есть свой комментарий.
